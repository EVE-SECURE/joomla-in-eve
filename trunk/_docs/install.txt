1. Get fresh install of Joomla 1.5 here: http://www.joomla.org/download.html
2. Unpack it to directory on your local test machine
3. Install it and delete installation folder
	(use default jos_ prefix, otherwise you'll have to change prefix in .sql commands)
4. Get EVE MySQL db dump here: http://eve.no-ip.de/qr100/ (or check eve-id.net wiki, or google it)
5. Import it to joomla database (strongly recommend to use commandline)
6. Check-out joomla-in-eve from SVN into separate folder and use it as project root

7. link files and directories
	A) Linux
	component folders:
	ln -s <joomla-in-eve root>/com_eve/admin <joomla root>/administrator/components/com_eve 
	ln -s <joomla-in-eve root>/com_eve/site <joomla root>/components/com_eve 
	
	ln -s <joomla-in-eve root>/com_evechartracking/admin <joomla root>/administrator/components/com_evechartracking 
	ln -s <joomla-in-eve root>/com_evechartracking/site <joomla root>/components/com_evechartracking 

	language files:
	ln -s <joomla-in-eve root>/com_eve/admin/languages/en-GB.com_eve.menu.ini <joomla root>/administrator/language/en-GB/en-GB.com_eve.menu.ini 
	ln -s <joomla-in-eve root>/com_eve/admin/languages/en-GB.com_eve.ini <joomla root>/administrator/language/en-GB/en-GB.com_eve.ini 
	ln -s <joomla-in-eve root>/com_eve/site/languages/en-GB.com_eve.ini <joomla root>/language/en-GB/en-GB.com_eve.ini 

	ln -s <joomla-in-eve root>/com_eve/admin/languages/en-GB.com_evechartracking.menu.ini <joomla root>/administrator/language/en-GB/en-GB.com_evechartracking.menu.ini 
	ln -s <joomla-in-eve root>/com_eve/admin/languages/en-GB.com_evechartracking.ini <joomla root>/administrator/language/en-GB/en-GB.com_evechartracking.ini 
	ln -s <joomla-in-eve root>/com_eve/site/languages/en-GB.com_evechartracking.ini <joomla root>/language/en-GB/en-GB.com_evechartracking.ini 

	template directory:
	ln -s <joomla-in-eve root>/tpl_eve_igb <joomla root>/templates/eve_igb
	
	plugin files:
	ln -s <joomla-in-eve root>/plg_system_eve/eve.php <joomla root>/plugins/system/eve.php
	ln -s <joomla-in-eve root>/plg_system_eve/eve.xml <joomla root>/plugins/system/eve.xml
	ln -s <joomla-in-eve root>/plg_system_eve/en-GB.plg_system_eve.ini <joomla root>/administrator/language/en-GB/en-GB.plg_system_eve.ini
	
	ln -s <joomla-in-eve root>/plg_user_eve/eve.php <joomla root>/plugins/user/eve.php
	ln -s <joomla-in-eve root>/plg_user_eve/eve.xml <joomla root>/plugins/user/eve.xml


	B) Windonws XP
	Unfortunately, Windows does not support links to files, but only for directories.
	Use some tool to create juction points to directories (e.g. http://www.rekenwonder.com/linkmagic.htm)
	for components and template. Copy rest files to appropriate location 
	
	C) Windows Vista
	Honestly, I've got no idea.... 
	
8. Execute SQL commands to create database tables, insert menu items and include plugins
